Set Break Point & Confirm Pools

Each district gets a Development Character Score. Higher score = more urban/developed. The break point divides them into Pool A (stronger) and Pool B (weaker).

How to use: Districts are ranked by score from highest to lowest. You draw a line in this ranking — districts above the line go to Pool A, below go to Pool B. The tool suggests the best natural break (the biggest gap in scores) but you can adjust freely using the slider, by typing a score, or by clicking any district row directly.
